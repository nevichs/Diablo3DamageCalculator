Everything is self contained in the Diablo 3 Damage Calculator.

TO RUN
- Double Click on the executable jar file

Features
-New Character: select a class and level, have to input all the gear
-Import from Armory: select an existing character and import all their gear, passive skills, level, and etc.
-Load from a save: load from a perviously saved character. 
	Also can delete by right clicking on the load select box after a load file has been click on.

Saving
-Saving is available on the main calculation screen. This will create a folder called "saves" in the directiory the jar file is in.
-Will ask to replace if the chosen name conflicts with an existing one.

Notes about Calculation
-Does account for set bonus
-Will account for active skills if those are chosen from the skill list.
